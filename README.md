# Business Sales Performance Analytics

## Objective
Analyze sales data to identify revenue trends, top-selling products, category performance, and regional performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Findings
- January generated the highest revenue.
- Laptop was the top-selling product.
- Technology category contributed the highest sales.
- West region achieved the highest revenue.

## Conclusion
This analysis identified important sales trends and business opportunities for growth.
